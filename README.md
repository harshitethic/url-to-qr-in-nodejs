<h2>URL TO QR Generator in nodejs</h2>
<p>This script generates a QR code in the terminal for a given URL input using Node.js and qrcode package.</p>
<h3>Installation:</h3>
<ul>
  <li>Clone the repository: <code>git clone https://github.com/harshitethic/url-to-qr-nodejs.git</code></li>
  <li>Install dependencies: <code>npm install</code></li>
</ul>
<h3>Usage:</h3>
<ul>
  <li>Run the script: <code>npm run url-to-qr-in-nodejs</code></li>
  <li>Enter a valid URL when prompted.</li>
  <li>A QR code will be generated and displayed in the terminal.</li>
</ul>
